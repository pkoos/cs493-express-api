## API calls and requirements

### Summary

The base path for all API calls is `/api/v1`.

* `/business/add`
* `/business/modify/:id`
* `/business/remove/:id`
* `/businesses`
* `/businesses/:id`
* `/business/:id`
* `/review/add`
* `/review/modify/:id`
* `/review/remove/:id`
* `/photo/add`
* `/photo/remove/:id`
* `/photo/modify/:id`
* `/photos/:id`
